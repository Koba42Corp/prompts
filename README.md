## ğŸš€ Quick Start

### Using Prompts

```bash
# Clone Koba42's repository
git clone https://github.com/koba42/prompts.git

# Browse prompts
cd prompts/prompts
# Find and copy any prompt template
```

### Running Code Examples

```bash
# Navigate to scripts directory
cd scripts

# Install dependencies
npm install

# Set up your API key
echo "OPENAI_API_KEY=your_key_here" > .env

# Run any example
node 01-basic-setup.js
```

### Reading System Documentation

```bash
# Navigate to systems directory
cd systems

# Read any system guide
cat mongodb.md
cat supabase.md
```
